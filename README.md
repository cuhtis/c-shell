# shell
* A simple shell program written in C
* Made by Curtis Li
* Note: Taking name suggestions

## Features
* Command line parsing
* Program foreground execution
* Builtins: cd, exit, pwd
* AND, OR support (&&, ||)
* Input and output redirection (<, >, 2>, &>)
* Pipes (|)
